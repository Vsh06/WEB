Program 4: Frequency of Words

Problem Statement: Develop a python to print 10 most frequently appearing words in a text file. Use dictionary with distinct words and their frequency of occurrences. [cite_start]Sort the dictionary in reverse order of frequency and display dictionary slice of first 10 items[cite: 68, 69, 70]. 
import operator # for dictionary sorting
import itertools # for dictionary slicing

fname = input("enter the file name:") # Step 1: ask for file name
try:
    fhand = open(fname) # Step 2: open the file
    d1 = dict()
    
    # Step 3: read each line and split into words
    for line in fhand:
        words = line.split()
        for word in words:
            if word in d1:
                # Step 4: count each word
                d1[word] = d1[word] + 1
            else:
                d1[word] = 1

    # Step 5: print the original dictionary
    print("\n Original dictionary:")
    print(d1)

    # Step 6: sort dictionary by frequency (descending order)
    print("\n The reverse ordered sorted dictionary ")
    sorted_d = dict(sorted(d1.items(), key=operator.itemgetter(1), reverse=True))
    print(sorted_d)

    # Step 7: slice the sorted dictionary (top 10 words)
    new_dict = dict(itertools.islice(sorted_d.items(), 10))
    print("\n Dictionary slice of first 10 items:", new_dict)

except:
    print("file name not found")

Program 5: Bubble Sort Marks

Problem Statement: Develop a program to read 6 subject marks from the keyboard for a student. [cite_start]Generate a report that displays the marks from the highest to the lowest score attained by the student using the Bubble Sort technique[cite: 2, 3, 4]. 
marks = [] # 1-D list
# Reading 6 subject marks
for i in range(6):
    m = int(input("Enter mark {}: ".format(i+1)))
    marks.append(m)

# Bubble Sort (Descending order)
n = len(marks)
for i in range(n):
    for j in range(0, n-i-1):
        if marks[j] < marks[j+1]: # change < for descending
            marks[j], marks[j+1] = marks[j+1], marks[j]

# Display sorted marks
print("\nMarks from Highest to Lowest:")
for m in marks:
    print(m)

Program 6: Sort File Contents

[cite_start]Problem Statement: Develop a program to sort the contents of a text file and write the sorted contents into a separate text file[cite: 42]. 
fname = input('Enter file name/directory: ')
with open(fname, 'r') as f:
    read = f.read()
    word = read.split()
    l = []
    for i in word:
        l.append(i)
    l.sort()

wname = input('Enter file name to create: ')
with open(wname, 'w') as w:
    for j in l:
        w.write(j)
        w.write('\n')
        
print("File\t" + wname + "\tCreated Successfully")
